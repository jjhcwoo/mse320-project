# MSE 320 - Machine Design
**Project: Merry-Go-Round**

**Group 13**

The objective of this project is to design and develop a safe, mechanical merry-go-round, intended for children aged six and under. The purpose of this project is to create a functional and reliable design that encourages outdoor physical activity, supporting healthy child development.
